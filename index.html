Discord Unix Timestap Table

This is where I set my stream schedule information. It's automatically converts date and time into unix timestamp code. Why do this? I like to post my schedule on discord but it's so hard to do manually (converted each date into unix timecode), so now you know. Feel free to use this for your server as well!

Input Stream Title
Autofill *This enable auto-fill on other input dates, applies when the first input date is specified.
